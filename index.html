    How to use this builder
    1. Add blocks in the order you want to match text.
    2. Set a quantifier for each block (optional, one, many, ranges).
    3. Enable flags if needed (see descriptions below).
    4. Paste text into Tester to see live matches.
    5. Use Copy regex to copy the final expression.

    What flags mean

    • i — case-insensitive, e.g. cat matches Cat.
    • m — multiline mode, so ^ and $ work on each line.
    • s — dotall mode, so . can match newline characters too.
    • g — global search, finds all matches instead of just the first one.

    Block quick reference

    • \d digit, \w word char, \s whitespace.
    • ^ start of text/line, $ end of text/line.
    • . any character (except new line unless s is enabled).
    • | alternation (either left or right side matches).
